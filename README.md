## Features

- Browse SpaceX rockets with real-time data
- View detailed rocket specifications
- Custom-designed UI with glassmorphic effects
- Async/Await for smooth performance
- MVVM architecture pattern

## Technologies

- **SwiftUI** - UI framework
- **Swift** - Programming language
- **Async/Await** - Concurrency
- **MVVM** - Architecture pattern

## Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

## Installation

1. Clone the repository
```bash
git clone https://github.com/ninikurshavishvili/spaceXProject.git
```

2. Open the project
```bash
cd spaceXProject/spaceXApp
open spaceXApp.xcodeproj
```

3. Build and run (Cmd + R)

## API

Uses [SpaceX API v4](https://api.spacexdata.com/v4/rockets) - no API key required.
